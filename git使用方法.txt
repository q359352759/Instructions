去github上创建一个仓库
进入文件夹 	git init 初始化
git remote add origin 加上github上面的什么名字
开始提交
git add .      	//将所有的本地文件提交到缓存中；后面的 . 代表所有文件；
git commit -m "commit all file"  //将所有文件提交到本地库中
git push -u origin master //第一次提交 添加命令参数 -u 确保关联本地库和远程库
git push origin master 		//非第一次提交使用此命令即可
